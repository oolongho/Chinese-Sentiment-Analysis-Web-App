# Chinese-Sentiment-Analysis-Web-App

研究内容

## 1、数据采集与预处理
开发京东商品评论爬虫，采集 3000 评论数据，其中 2500 构建语料库和训练模型，500 用于测试准
确度。
设计数据清洗流水线：去重、去 HTML 标签、繁简转换、敏感信息过滤。
实现 Jieba 自定义词典分词与 LTP 句法分析，构建电商领域情感词典。

## 2、双通道情感分析引擎
情感词典通道：基于知网词典构建扩展词典，引入程度副词、否定词规则，实现可解释性模型。
深度学习通道：微调 chinese-roberta-wwm-ext 模型，采用交叉熵损失，对比 LSTM 基准模型。
语音识别通道：集成 SenseVoiceSmall 模型，实现低延迟音频转文字（支持中英混输）。

## 3、轻量化模型优化与部署
模型服务化：FastAPI 异步处理，支持批量请求与流式响应。
性能目标：单条文本分析<50ms，音频转写<1.5x 实时率。

## 4、Web 应用开发
前端：React+TypeScript+Vite 等
音频交互：Wavesurfer.js 实现波形可视化、区域选择与实时播放。
后端：FastAPI 提供 RESTful API，处理文本/音频上传、结果缓存。
实现功能：单文本分析、批量文件上传、性能指标统计（准确率、F1、响应时间）。
